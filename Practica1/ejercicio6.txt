a- El sistema de archivos es el encargado de organizar la forma en que se guardan los archivos en dispositivos 
de almacenamiento.

b- Los soportados por GNU/Linux son los extended v2, v3 y v4.

c-  Si es posible.

d- La estructura basica de los file system en GNU/Linux es:
    / -> Es el tope de la estructura de directorios.
    /home -> Se almacenan los archivos del usuario.
    /boot -> Archivos de arranque.
    /var -> Informacion que varia segun el tamaño.
    /etc -> Archivos de configuracion.
    /bin -> Archivos binarios y ejecutables.
    /deb -> enlace a dispositivos.
    /usr -> plicaciones de usuario.
    Las siglas FHS significan FileSystem Hierarchy Standard.
